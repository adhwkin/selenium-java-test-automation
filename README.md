# Selenium Java Automation Framework

This project demonstrates a test automation framework built using Selenium WebDriver with Java.

## Tech Stack

- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)
- Extent Reports
- Log4j2 Logging
- Data Driven Testing (Excel)
- Parallel Test Execution

## Features

- ThreadLocal WebDriver (parallel execution ready)
- Page Object Model (POM)
- Explicit waits for stability
- Retry mechanism for flaky tests
- Soft assertions for better validation
- Screenshot capture on failure

## Test Scenario

Login functionality automation for:

https://the-internet.herokuapp.com/login

## How to Run Tests
mvn clean test
